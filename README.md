### **📌 Featured Projects**

#### **[Assistente Salute 🏥](https://github.com/Saalamandras/assistente-salute)**

**Chatbot AI per triage ortopedico in italiano — Progetto portfolio per ruoli in NLP/AI Engineering**

**Descrizione:**  
Assistente Salute è un chatbot conversazionale che simula un flusso di triage ortopedico per utenti anziani. Il bot:

- Accoglie l’utente con un disclaimer chiaro e non burocratico.
- Pone domande contestuali sui sintomi muscoloscheletrici.
- Fornisce risposte orientative e raccomanda lo specialista più adatto.
- Gestisce la prenotazione di appuntamenti su un calendario mock.

**Stack Tecnico:**


| Componente | Tecnologia                          |
| ---------- | ----------------------------------- |
| LLM        | Anthropic Claude (claude-3-5-haiku) |
| SDK        | `anthropic` Python SDK              |
| UI         | Gradio                              |
| Deploy     | Hugging Face Spaces                 |
| Config     | python-dotenv                       |


**Punti Chiave:**

- **Prompt Engineering**: Il file `system_prompt.py` è il cuore del progetto, progettato per definire tono, stile, e vincoli in italiano, con regole di safety esplicite.
- **Prototipo Dimostrativo**: Progetto sviluppato come artefatto per candidature in ruoli AI/NLP engineering.
- **Miglioramenti Futuri**: Integrazione con Google Calendar API, autenticazione utente, e streaming delle risposte.

**Come Eseguire in Locale:**

```bash
git clone https://github.com/Saalamandras/assistente-salute
cd assistente-salute
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -r requirements.txt
python app.py
```
